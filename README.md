🚀 GitHub Webhook Event Tracker

A real-time GitHub webhook monitoring system built using Flask, MongoDB, and JavaScript polling.

This application listens to GitHub repository events (Push, Pull Request, Merge), stores them in MongoDB, and displays the latest activity in a live-updating UI.

📌 Project Overview

This project demonstrates:

Backend webhook handling using Flask

Secure event processing

MongoDB integration

REST API design

Frontend polling mechanism

Clean project architecture using Blueprints

The UI automatically updates every 15 seconds to show the latest repository activity.

🏗️ Tech Stack

Backend: Python + Flask

Database: MongoDB

Frontend: HTML + JavaScript

Version Control: Git & GitHub

Testing: Postman / GitHub Webhooks
